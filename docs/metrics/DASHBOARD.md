# Competitiveness Dashboard

Maintainer-facing metrics for adoption and ecosystem tracking.

## North-star
- `apps_in_production`

## Adoption metrics
- `monthly_active_apps`
- `retention_90d`

## Quality metrics
- `ci_pass_rate`
- `regression_detection_rate`
- `time_to_fix_critical_bug_hours`

## Ecosystem metrics
- `active_contributors`
- `plugin_count`
- `plugin_certified_count`

## Collection cadence
- weekly for CI and regression metrics
- monthly for adoption and ecosystem metrics

## Data contract

Use `docs/metrics/metrics.schema.json` for machine-validated payloads.
